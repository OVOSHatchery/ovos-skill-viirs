## About

Near real time Satellite imagery from NASA's [Visible Infrared Imaging Radiometer Suite](https://wiki.earthdata.nasa.gov/display/GIBS/2015/12/10/VIIRS+is+Here)
  
Can be used as idle screen for the mark2
